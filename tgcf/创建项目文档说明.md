## 🚀 生成PDF文档

### 方法一：使用Puppeteer自动生成（推荐）

```bash
# 确保已安装Node.js
node --version

# 安装Puppeteer依赖（如果需要）
npm install puppeteer

# 运行PDF生成脚本
cd /var/www/tgcf
node generate-pdf.js
```

### 方法二：手动转换（备选方案）

```bash
# 生成HTML版本
cd /var/www/tgcf
node convert-to-pdf.js

# 然后使用浏览器打开HTML文件，选择"打印"→"另存为PDF"
```

### 方法三：使用在线工具转换

1. 打开 `项目技术文档.md` 文件
2. 复制内容到在线Markdown转PDF工具
3. 下载生成的PDF文件
